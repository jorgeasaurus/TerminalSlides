Import-TerminalPresentation

Description

Load a presentation from PSD1, JSON, Markdown, or Markdown-compatible content.

Examples

Example 1

$jsonPath = Join-Path `
  ([IO.Path]::GetTempPath()) 'terminalslides-deck.json'
Export-TerminalPresentation `
  -Presentation $deck -Format Json -Path $jsonPath | Out-Null

$importedDeck = Import-TerminalPresentation -Path $jsonPath

Parameters

-Path

Specifies the Path value.

Type
System.String
Required
true
Position
0
Pipeline input
false

Syntax

Import-TerminalPresentation -Path <String>
