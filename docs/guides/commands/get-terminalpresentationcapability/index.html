Get-TerminalPresentationCapability

Description

Inspect terminal dimensions, color, Unicode, graphics, redirection, and interactivity support.

Examples

Example 1

Get-TerminalPresentationCapability |
  Format-List HostName, Width, Height, `
    TrueColorSupport, UnicodeSupport, Interactive

Parameters

This command has no command-specific parameters.
