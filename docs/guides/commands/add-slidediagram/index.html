Add-SlideDiagram

Description

Build a lightweight node-and-edge diagram with the diagram DSL.

Examples

Example 1

$deck | Add-TerminalSlide -Title 'Flow' -Content {
  Add-SlideDiagram -Content {
    Add-SlideDiagramNode -Id source -Label 'Source'
    Add-SlideDiagramNode -Id deploy -Label 'Deploy'
    Add-SlideDiagramEdge -From source -To deploy -Label 'publish'
  }
} | Out-Null
