Add-SlideTable

Description

Format PowerShell objects as a table within the selected slide region.

Examples

Example 1

$rows = @(
  [pscustomobject]@{ Service = 'API'; Status = 'Healthy' }
  [pscustomobject]@{ Service = 'Worker'; Status = 'Healthy' }
)
$deck | Add-TerminalSlide -Title 'Services' -Content {
  Add-SlideTable -Data $rows -Border
} | Out-Null
