Move-TerminalSlide

Description

Move a slide to a new one-based position and reindex the deck.

Examples

Example 1

$deck | Move-TerminalSlide `
  -Index 4 `
  -DestinationIndex 2 | Out-Null

Parameters

-DestinationIndex

Specifies the DestinationIndex value.
