Add-SlideCode

Description

Render a syntax-highlighted code block with an optional border.

Examples

Example 1

$deck | Add-TerminalSlide -Title 'Code' -Layout CodeFocus -Content {
  Add-SlideCode `
    -Code "Get-Process | Sort-Object CPU -Descending" `
    -Language powershell `
    -Region Code `
    -Border
} | Out-Null
