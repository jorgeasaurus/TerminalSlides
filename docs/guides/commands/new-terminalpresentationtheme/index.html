New-TerminalPresentationTheme

Description

Create and register a reusable truecolor theme for the current session.

Examples

Example 1

New-TerminalPresentationTheme `
  -Name Ocean `
  -Background '#06131F' `
  -Foreground '#E7F5FF' `
  -Primary '#38BDF8' `
  -Accent '#A78BFA' | Out-Null

$deck = New-TerminalPresentation `
  -Title 'Ocean deck' -Theme Ocean
