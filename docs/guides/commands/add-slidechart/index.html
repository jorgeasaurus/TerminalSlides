Add-SlideChart

Description

Visualize labeled values as bars, a line, sparkline, or gauge.

Examples

Example 1

$data = @(
  [pscustomobject]@{ Label = 'Build'; Value = 82 }
  [pscustomobject]@{ Label = 'Test'; Value = 96 }
)
$deck | Add-TerminalSlide -Title 'Pipeline' -Content {
  Add-SlideChart -Data $data `
    -ChartType HorizontalBar `
    -Title 'Success rate'
} | Out-Null
