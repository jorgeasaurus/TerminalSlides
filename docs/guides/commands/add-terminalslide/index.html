Add-TerminalSlide

Description

Append a slide through the pipeline and compose its elements with the slide DSL.

Examples

Example 1

$deck | Add-TerminalSlide `
  -Title 'Agenda' `
  -Layout TitleAndContent `
  -Content {
    Add-SlideBullet 'Architecture'
    Add-SlideBullet 'Delivery plan'
  } | Out-Null
