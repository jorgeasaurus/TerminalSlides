Export-TerminalPresentation

Description

Write a deck as ANSI, plain text, Markdown, HTML, PSD1, or JSON.

Examples

Example 1

$exportPath = Join-Path `
  ([IO.Path]::GetTempPath()) 'terminalslides-deck.html'

Export-TerminalPresentation `
  -Presentation $deck `
  -Format Html `
  -Path $exportPath
