Start-TerminalSlidesDemo

Description

Launch the TerminalSlides feature tour or the Intune Hydration Kit showcase, or return either deck for inspection.

Examples

Example 1

# Uses Sixel images when supported, with a block fallback.
Start-TerminalSlidesDemo
Start-TerminalSlidesDemo -Name IntuneHydrationKit

# Force terminal-cell image rendering instead.
Start-TerminalSlidesDemo -ImageRenderer Blocks

# Inspect a showcase without presenting it:
$demo = Start-TerminalSlidesDemo `
  -Name IntuneHydrationKit -PassThru
