Add-SlideNotes

Description

Attach private speaker notes to the slide currently being composed.

Examples

Example 1

$deck | Add-TerminalSlide -Title 'Roadmap' -Content {
  Add-SlideText 'Three milestones lead to launch.'
  Add-SlideNotes `
    'Pause here and ask for questions.'
} | Out-Null

Parameters

-Text

Specifies the Text value.

Type
System.String
Required
true
Position
0
Pipeline input
false

Syntax

Add-SlideNotes -Text <String>
