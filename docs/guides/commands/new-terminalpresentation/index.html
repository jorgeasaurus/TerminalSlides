New-TerminalPresentation

Description

Create an empty presentation and set its title, author, theme, and viewport defaults.

Examples

Example 1

$deck = New-TerminalPresentation `
  -Title 'Quarterly engineering review' `
  -Subtitle 'Platform reliability' `
  -Author 'Engineering' `
  -Theme Midnight
