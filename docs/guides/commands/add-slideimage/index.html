Add-SlideImage

Description

Render a local image as truecolor terminal cells with accessible fallback text.

Examples

Example 1

$imagePath = Join-Path `
  (Get-Module TerminalSlides).ModuleBase `
  'Assets/presentation-team-photo.jpg'

$deck | Add-TerminalSlide `
  -Title 'Team rehearsal' `
  -Layout ImageFocus `
  -Content {
    Add-SlideImage `
      -Path $imagePath `
      -AltText 'Three engineers rehearsing a presentation' `
      -Region Image
  } | Out-Null
