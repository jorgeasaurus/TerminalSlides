Guides / Command reference

Test-TerminalPresentation

Description

Check every slide for overflow at one or more terminal viewport sizes.

Examples

Example 1

$results = Test-TerminalPresentation `
  -Presentation $deck `
  -Viewport '80x24', '120x35'

$results | Format-Table Viewport, Slide, Fits

Parameters

-Presentation

Specifies the Presentation value.

Type

true

false

-Viewport
