Add-SlideQuote

Description

Add a centered quotation with optional attribution.

Examples

Example 1

$deck | Add-TerminalSlide -Title 'Principle' -Layout Quote -Content {
  Add-SlideQuote `
    -Text 'Make the change easy, then make the easy change.' `
    -Attribution 'Kent Beck' `
    -Region Quote
} | Out-Null
